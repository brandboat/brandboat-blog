**這問題是因為在CompizConfig Setting Manager中Gereral欄位中開啟Mipmap選項的緣故。**
<br/><br/>
![markdown](http://i292.photobucket.com/albums/mm16/namtlade/Windowswitcherfail.png)<br/><br/>
圖片來源自: <http://ubuntuforums.org/showthread.php?p=6024749>

###解決方法如下 ： 

- **開啟 CompizConfig Setting Manager，並搜尋 Application Switcher.**

<a href="http://www.flickr.com/photos/keithworld/12163523303/" title="select003, on Flickr"><img src="http://farm4.staticflickr.com/3690/12163523303_c7245bebb0_z.jpg" alt="Bird Nest"></a>

- **關閉 Mipmap 選項，並開啟 Enable Application Switcher.**

<a href="http://www.flickr.com/photos/keithworld/12163521963/" title="select005, on Flickr"><img src="http://farm6.staticflickr.com/5473/12163521963_313a182365_z.jpg" alt="Bird Nest"></a><br/><br/>
參考資料 ： <http://ubuntuforums.org/showthread.php?p=6024749>
